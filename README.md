# orbita-circular-en-python
# Órbita Circular en Python

Este proyecto muestra una representación simple de una órbita circular en dos dimensiones utilizando Python. Emplea las librerías NumPy y Matplotlib para calcular y visualizar la trayectoria de un objeto a partir de un radio definido y un conjunto de ángulos.

## Descripción
El programa calcula las coordenadas X e Y de una órbita circular usando funciones trigonométricas. El radio utilizado es de 6771 km, valor cercano al radio de una órbita baja terrestre, lo que permite una visualización más realista del movimiento orbital.

## Cómo funciona
- Se genera un arreglo de ángulos entre 0 y 2π.
- Se calculan las coordenadas X e Y usando coseno y seno.
- Se grafica la órbita manteniendo la escala real con `axis('equal')`.
- Se muestran títulos y etiquetas para una mejor comprensión del gráfico.

## Características
- Visualización clara de una órbita circular en 2D.
- Uso de NumPy para cálculos matemáticos.
- Uso de Matplotlib para gráficos científicos.
- Código simple y fácil de entender, ideal para principiantes.

## Tecnologías
- Python 3
- NumPy
- Matplotlib

## Objetivo del proyecto
Este proyecto tiene fines educativos y está pensado para personas que están comenzando en Python o que desean aprender conceptos básicos de visualización científica y movimiento orbital de forma simple y visual.



# Circular Orbit in Python

This project shows a simple two-dimensional representation of a circular orbit using Python. It uses the NumPy and Matplotlib libraries to calculate and visualize the trajectory of an object based on a defined radius and a set of angles.

## Description
The program calculates the X and Y coordinates of a circular orbit using trigonometric functions. The chosen radius is 6771 km, close to a low Earth orbit radius, allowing a more realistic visualization of orbital motion.

## How it works
- An array of angles between 0 and 2π is generated.
- X and Y coordinates are calculated using cosine and sine.
- The orbit is plotted while keeping the correct scale with `axis('equal')`.
- Titles and axis labels are added for clarity.

## Features
- Clear 2D visualization of a circular orbit.
- Use of NumPy for mathematical calculations.
- Use of Matplotlib for scientific plotting.
- Simple and beginner-friendly code.

## Technologies
- Python 3
- NumPy
- Matplotlib

## Project goal
This project is for educational purposes and is intended for people who are starting with Python or want to learn basic scientific visualization and orbital motion concepts in a simple and visual way.
